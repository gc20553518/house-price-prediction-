# House Price Prediction (Kaggle Project)

## Overview
This project builds regression models to predict housing prices using a real-world dataset.  
The goal is to improve prediction accuracy through data preprocessing, feature engineering, and model evaluation.

## Results
- Kaggle Ranking: Top 15  
- RMSE: 1,184,578  

## Methods
- Data cleaning (handling missing values and outliers)  
- Log transformation of target variable (log(SalePrice))  
- Feature engineering (TotalSF, TotalBath)  
- Regression modeling  
- Model diagnostics (residual analysis, VIF)  

## Tools
- R  
- ggplot2, dplyr  

## Workflow
Data → Feature Engineering → Modeling → Evaluation → Submission
